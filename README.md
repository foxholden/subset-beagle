# subset-beagle
Fast subsetting of **ANGSD/Beagle** genotype likelihood (`.beagle` / `.beagle.gz`) files.  



## Usage

### **Basic command**
```bash
subset_beagle --input input.beagle.gz --output output.beagle.gz

Keep only selected individuals
```bash
subset_beagle \
    --input input.beagle.gz \
    --keep sample_list.txt \
    --output subset.beagle.gz
```
Remove selected individuals
```bash
subset_beagle \
    --input input.beagle.gz \
    --remove sample_list.txt \
    --output subset.beagle.gz
```

Flags 
Flag	Long Flag	Description
-i	--input	Input BEAGLE or BEAGLE.GZ file
-o	--output	Output BEAGLE or BEAGLE.GZ file
-k	--keep	Text file containing individuals to keep (one per line, matching BEAGLE header names)
-r	--remove	Text file containing individuals to remove (one per line)


Input format
The --keep or --remove file should contain one sample ID per line, matching the sample names found in the BEAGLE header, e.g.:

Ind0
Ind1
Ind152

Notes

Compatible with BEAGLE files generated by ANGSD (-doGlf 2)

No need to regenerate GLs to drop individuals â€” this tool operates entirely on the text-based BEAGLE format

Output retains the original ordering of remaining individuals
